Class HashPMap<K, V>

Can use anything as key, using the provided hash function as internal unique identifier, meaning:

const hash = k => k.id;
const map = new HashPMap<object, string>(hash);
const k = { id: 1 };
map.set(k, 'some value');
map.has(k); // true
map.has({ id: 1 }); // true
map.has({ id: 2 }); // false

  • K - type of keys
