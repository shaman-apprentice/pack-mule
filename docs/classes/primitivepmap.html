Class PrimitivePMap<K, V>

PrimitivePMap is basically a HashPMap for Primitives only, with predefined hash function, which provides out of the box collision protection for e.g. 1 and "1" or undefined and "undefined".

Type parameters

  • K - type of keys - `K` must extend {@link Primitive}
  • V - type of values

Implements

  • Iterable<IEntry<K, V>>
