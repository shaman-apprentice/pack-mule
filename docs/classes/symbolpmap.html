Class SymbolPMap<K, V>

Can use Objects and functions as keys. The keys work by reference, meaning:

const map = new SymbolPMap<object, string>();
const k = {};
map.set(k, 'some value');
map.has(k); // true
map.has({}); // false

If you want the behavior of the key to be value-like see HashPMap.

SymbolPMap makes it work internally through adding (and removing in case of removal) a Symbol to the key object. This does not affect the normal behavior of your key like e.g. in Object.keys(key). Here is a nice and easy to read article about Symbols, if you are interested in learning more about Symbols.
